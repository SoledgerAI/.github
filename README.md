# SoledgerAI

SoledgerAI builds AI-enabled systems for governed, domain-driven software generation.

We believe serious AI software should not begin with code.

It should begin with:

```txt
truth
state
authority
events
observability
tests
```

Our work focuses on turning business rules, workflows, controls, and domain knowledge into structured application packages that AI coding agents can safely implement.

---

## What we are building

SoledgerAI is developing an AI-native software factory approach built around:

* application constitutions
* domain models
* invariants
* policy templates
* state machines
* event models
* observability requirements
* test plans
* agent handoff protocols

The goal is to reduce hallucinated business logic, undocumented assumptions, brittle code generation, and ungoverned automation.

## Featured project

## Operating philosophy

AI should accelerate software delivery, but it should not invent the business.

Humans define the truth.

Agents scaffold, validate, generate, test, and report gaps.

That separation is the foundation of responsible AI-native application development.
